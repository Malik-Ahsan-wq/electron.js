# Assets

Place your app icons here before running `npm run dist`:

- `icon.ico`  — Windows icon (256x256, ICO format)
- `icon.icns` — macOS icon (ICNS format)
- `icon.png`  — Linux icon (512x512 PNG)

You can generate these from a single 1024x1024 PNG using tools like:
- https://www.icoconverter.com/ (for .ico)
- `electron-icon-builder` npm package
